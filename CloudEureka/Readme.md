# Spring Cloud Eureka 프로젝트  

## 스프링 클라우드 유레카서버  

 - msa 클라이언트 관리 모니터링 하는 서버 역할
 - 30초 단위로 클라이언트의 핑정보를 수집하여 모니터링함.

 
## Discovery Pattern
 마이크로 서비스 아키텍처는 일반적으로 클라우드 환경에서 클라우드에 올라가는 인스턴스는 네트워크 위치가 동적으로 생성이 되는 경우가 많다.
 또 인스턴스 추가, 삭제, 오토스케일링 등의 이유로 IP가 동적으로 변경될 수 있는 부분에 대응하기 위해 MSA에서 디스커버리 패턴을 활용하게 되었다.
 
 
 
 - @EnableEurekaServer 유레카 서버임을 선언
 - @EnableDiscoveryClient 유레카 클라이언트 선언
